# Pathway Bridge Suite

Professional WordPress connectivity hub for Forms, Posts, and Routes.

## Features
- **Forms Bridge**: Capture and route form submissions (Elementor, CF7, etc.).
- **Posts Bridge**: Sync Custom Post Types with external APIs.
- **Routes Bridge**: Create custom WP REST API endpoints with custom PHP logic.
- **Workflow Engine**: Chainable jobs for data transformation and delivery.

## Installation
1. Upload to `/wp-content/plugins/`.
2. Activate via WordPress Admin.
3. Configure bridges in the Pathway Dashboard.
